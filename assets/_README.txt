GENERATING ASSETS

All assets must be stored in the assets/ directory, the source files for the assets must be stored in assets/_src/
Export guides below are intended for Adobe Illustrator 2024:

- Sheet item icons should be .png images, 95x95@8ppi
	• check "use dartboards"/"all" when exporting
	• set resolution to 8 ppi
	• select "Art Optimized (supersampling)
